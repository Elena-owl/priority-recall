# Session Start Workflow

Follow this workflow at the start of each session to recall priority memories.

## Step 1: Load Recent Sessions

Read the last 3-5 session files from `memory/priority-recall/`

## Step 2: Extract Priority Items

Pull items with score ≥ 3 into a quick-scan format.

## Step 3: Present Quick Scan

Format for quick human scanning:

```
📌 MUST KNOW
• [Critical item] (score: 5)
• [Important item] (score: 4)

📎 RECENT CONTEXT
• [Normal item] (score: 3)

❓ NEEDS CLARIFICATION
• [Gap item from previous session]
```

## Step 4: Check for Gaps

Look for items marked "needs follow-up" in previous sessions. Present them as:
- "Previous session mentioned X - want me to clarify?"

## Step 5: Load by Tag (Optional)

If specific context is needed (e.g., "billing"), filter by tag:
```
Billing context:
- [item]
- [item]
```

---

**Tip:** Keep it brief. 5 items max for quick scan.
